# PagerSnapHelper
- RecyclerView를 viewpager처럼 사용하기 혹은 페이지 넘기기

```
val snapHelper = LinearSnapHelper() // Or PagerSnapHelper
snapHelper.attachToRecyclerView(recyclerView)``
```
